# 헬스케어포털 변경 이력 (Changelog)

이 문서는 헬스케어포털 프로젝트의 주요 변경 사항, 기능 추가 및 버그 수정 내역을 관리합니다.

## [2026-05-12]
### 기능 추가 및 구조 변경
- **추가 메뉴 동적 생성 기능**: 관리자 포털에서 고객사별 기본 메뉴 외에 새로운 커스텀 메뉴를 동적으로 생성 및 삭제할 수 있는 기능 추가
- **등급별 메뉴명 설정 기능**: 추가된 커스텀 메뉴의 이름을 전체 사용자 공통으로 설정하거나, 고객사 내 사용자 등급(Tier)별로 각각 다른 이름이 노출되도록 세분화하여 설정하는 UI 구현
- **사용자 포털 다이내믹 라우팅 및 렌더링 연동**: 로그인한 사용자의 등급 정보와 관리자가 설정한 커스텀 메뉴 데이터를 매칭하여 사용자 포털 GNB에 동적 메뉴가 즉시 반영되도록 로직 연동

## [2026-05-11]
### 변경 및 복구 사항
- **관리자 포털 안정화**: 코드 변경으로 인한 탐색 및 기능 회귀 문제 해결을 위해 20260423 백업을 기준으로 관리자 포털 복구
- **SPA 탐색 복구**: SPA(Single Page Application) 메뉴 탐색 무결성 검증 및 정상화

## [2026-04-29]
### 문서화
- **기능정의서 작성**: 기존 구현된 기능(동적 메뉴 관리 시스템, 등급별 서비스 노출 등)을 바탕으로 요구사항 정의서(`헬스케어포털_기능정의서.csv`) 생성

## [2026-04-17]
### 기능 추가 및 수정
- **관리자 메뉴 설정 복구**: 누락되었던 "고객사별 기본 메뉴명" 및 "노출/사용 설정" 기능 복구
- **관리자 서비스 에디터 구현**: 등급별 서비스 정보를 위한 시각적 에디터를 관리자 페이지 메인 레이아웃에 통합 완료 및 UI/UX 문제 해결

## [2026-04-14]
### UI/UX 개편 및 구조 변경
- **포털 UI 전면 개편**: "병원 정보" 메뉴 구조 재편, 상세 검색 필터(지역, 유형, 진료과) 구현 및 전체적인 레이아웃을 프리미엄 디자인으로 개선
- **동적 사이드바 네비게이션 구현**: 100개 이상의 사용자 등급을 처리할 수 있는 등급 코드 등록 시스템 및 재귀적 트리 설정 UI를 통한 등급별 가시성 제어 기능 구현
- **통합 버그 수정**: `app.js`의 코드 중복 및 구문 오류 해결을 통해 동적 메뉴 구성과 관리자 페이지의 등급별 가시성 설정이 정상적으로 연동되도록 수정
