# 헬스케어포털 기능 세부 정의서 (Detailed Feature Specifications)

본 문서는 헬스케어포털의 각 기능별 세부 동작 요건 및 비즈니스 로직을 정의합니다.

## 1. 사용자 포털 (User Portal)

### 1.4 사이드바 트리 네비게이션
*   **서브페이지 레이아웃**: 상단 GNB 메뉴 클릭 시, 좌측 사이드바와 우측 콘텐츠 영역으로 구성된 전문 서브페이지 레이아웃으로 전환됩니다.
*   **아코디언 트리뷰**: 
    *   사이드바에는 포털의 전체 메뉴 구조가 표시됩니다.
    *   사용자가 현재 위치한 카테고리의 브랜치만 자동으로 상세 확장(Expand)되며, 다른 메뉴는 요약된 상태로 유지되어 탐색 효율성을 높입니다.
*   **위치 가이드**: 상단에 '홈 > 메뉴 > 서브메뉴' 형태의 브레드크럼(Breadcrumb)을 제공하여 사용자 이동 경로를 직관적으로 안내합니다.

### 1.5 온라인 문의 (1:1 상담)
*   **비공개 상담 원칙**: 사용자는 본인이 작성한 문의 내역과 그에 대한 전문가의 답변만 확인할 수 있는 철저한 1:1 상담 환경을 가집니다.
*   **신속한 대응 고지**: "영업일 기준 3일 이내 답변" 원칙을 명시하여 사용자 기대치를 관리합니다.
*   **상태 추적**: '답변대기'와 '답변완료' 상태 배지를 통해 상담 진행 상황을 실시간으로 파악할 수 있습니다.

## 3. 기능 요구사항 매트릭스 (Summary)
| 분류 | 대분류 | 중분류 | 세부 요건 |
| :--- | :--- | :--- | :--- |
| **관리** | 브랜드 | 로고 | CI 이미지 파일 업로드 및 헤더 연동 (Base64) |
| **관리** | 브랜드 | 테마컬러 | P1(버튼/인증), P2(메뉴/CS) 이원화 설정 및 HEX 입력 |
| **관리** | 메뉴 | 구조 | 3Depth 재귀 트리 구성 및 노출 여부 토글 |
| **관리** | 서비스 | 문의관리 | 사용자 온라인 문의 내역 확인 및 전문가 답변 등록/수정 |
| **사용자** | 공통 | 헤더 | 호버 시 하위 메뉴가 전면 노출되는 메가메뉴 레이아웃 |
| **사용자** | 공통 | 사이드바 | 전체 메뉴 트리 구조 및 현재 위치 자동 확장 아코디언 적용 |
| **사용자** | 서비스 | 상담신청 | 상담 일자, 시간(09~17시), 메모를 입력하는 동적 신청 폼 |
| **사용자** | 서비스 | 온라인문의 | 1:1 비공개 문의 등록 및 전문가 답변 실시간 확인 |
| **사용자** | 공통 | 푸터 | 고객사/제공사 하이퍼링크 및 제휴 고지 문구 자동 생성 (상하 구분선 제거) |
| **사용자** | 공통 | 테마 | 관리자 설정 컬러 기반의 CSS 변수(--theme-color) 실시간 적용 |
